# TeamMembers_page
team page with member cards

## What's included
- Created /team route with a dedicated page
- Team member data stored in local JSON file (name, title, bio, photo)
- Cards mapped from JSON data with initials fallback if no photo
- Responsive layout: 3 columns desktop, 2 tablet, 1 mobile
- Auto-scroll animation with hover pause
- Hover overlay showing member bio
- Plain CSS Modules, no external libraries

## Folder structure
- app/team/page.tsx
- app/team/team.module.css
- components/TeamCard.tsx
- components/TeamCard.module.css
- data/team.json
- public/images/
